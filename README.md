## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Подключение Grafana
$ sudo -E bash -c 'docker-compose up'

При необходимости: mkdir $PWD/data

localhost:3010
